🔥 Autonomous Fire Fighting Robot
📌 Overview

This project is an autonomous robot designed to detect and extinguish fire. It uses flame sensors to identify fire and automatically moves toward it, activating a water pump to put it out.

⚙️ Components Used
Arduino Uno
L298N Motor Driver
Relay Module
3 Flame Sensors
4 DC Motors
Chassis + Wheels
Mini Water Pump
Pipe/Nozzle
7.4V / 12V Battery
Voltage Converters (12V to 5V)
Breadboard & Wires
⚡ Working Principle

Flame sensors detect fire and send signals to the Arduino.
The Arduino processes the input and controls the motors via the motor driver to move toward the fire.
Once detected, the relay activates the water pump to extinguish the fire.

🚀 Features
Automatic fire detection
Autonomous movement
Water-based fire extinguishing system
Low-cost and efficient
📉 Limitations
Limited range
Depends on battery life
Suitable for small-scale fires only
🔮 Future Improvements
Add camera for monitoring
Use AI for better detection
Increase range and efficiency
🙌 Conclusion

This project demonstrates a simple and effective fire-fighting robot that can reduce human risk and improve safety in small environments.
